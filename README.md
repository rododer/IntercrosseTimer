Eine Android-App für Intercrosse.

Entweder für den Zeitnehmer (Schiedsgericht):
Ein 30-Sekunden-Timer (Angriffszeit)
welcher zusätzlich 1 (Timeout), 2 (Viertelpause) oder 5 (Halbzeitpause) Minuten stoppen kann.

Für Feldschiedsrichter:
5-Sekunden-Timer (Zeit pro Spieler)

Der Timer kann über die Lautstärkewippe gesteuert werden:
- Leisertaste: Neustarten des Timers
- Lautertaste: Pausieren oder wieder starten des Timers

Durch Vibration kann der Timer haptisch ergänzt werden.

---------------- ENGLISH ----------------

An Android app for intercrosse.

For the timekeeper (official table):
A 30-second timer (attack clock)
which can additionally run 1 minute (timeout), 2 minutes (quarter break), or 5 minutes (halftime break).

For field referees:
A 5-second timer (time per player)

The timer can be controlled via the volume buttons:

Volume down: restart the timer

Volume up: pause or resume the timer

Haptic feedback can be added through vibration.
